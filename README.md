To start the app:
1. if you do not have yarn: ```$ brew install yarn```
1. ```$ yarn install```
1. ```$ yarn start```

In Xcode
1. Open scoreCard/ios/scoreCard.xcodeproj
1. select target device, either the simulator or a connected device
1. click run
